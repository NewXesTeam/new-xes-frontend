## npm 脚本

- `npm run build:dev` 构建项目（开发模式）
- `npm run build:prod` 构建项目（生产模式）
- `npm run prettier` 使用 Prettier 格式化代码

## 运行开发服务器

1. 下载 nginx
2. 复制 nginx 到根目录。_注意不要把 `conf/nginx.conf` 给覆盖了！！！_
3. 运行 nginx。（请先构建后再运行）
4. 访问 `http://127.0.0.1:8080`

## 关于使用

您可以直接使用我们提供的 登录功能 进行登录。
